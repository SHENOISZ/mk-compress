# mk-compress

A compressor written in C ++ for html, css and js.
Also compress css and js in inline html.


### How to use

```
~$mk-compress index.html

result: index.min.html
```

```
~$mk-compress style.css

result: style.min.css
```

```
~$mk-compress init.js

result: init.min.js
```
